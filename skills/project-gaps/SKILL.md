# Project Gaps (Deprecated)

`$project-gaps` preserves the legacy workflow while new prompts migrate to the
explicit names:

  $project-gaps`, read and follow `../project-gaps-implement/SKILL.md` completely.
- For Find-mode requests and all other `$project-gaps` invocations, read and
  follow `../project-gaps-find/SKILL.md` completely.

The selected canonical skill owns the workflow's scope, ledger contract,
delegation, and approval gates.
