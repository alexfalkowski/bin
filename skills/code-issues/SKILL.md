# Code Issues (Deprecated)

`$code-issues` preserves the legacy workflow while new prompts migrate to the
explicit names:

- For `Start <ID>`, `Approved <ID>-N`, `Done <ID>`, or `Implement
  $code-issues`, read and follow `../code-issues-implement/SKILL.md` completely.
- For Find-mode requests and all other `$code-issues` invocations, read and
  follow `../code-issues-find/SKILL.md` completely.

The selected canonical skill owns the workflow's scope, ledger contract,
delegation, and approval gates.
